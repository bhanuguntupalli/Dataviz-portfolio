# Dataviz-portfolio – SQL & Analytics Dashboard

## Problem Statement
Using SQL, BigQuery, and Looker Studio, create an end-to-end analytics solution that transforms raw eCommerce data into actionable business insights, moving beyond static reporting to highlight revenue drivers, customer behavior, and long-term value.

## What Was Completed
- Built analytical SQL views using CTEs and window functions to uncover revenue trends, category-level performance, customer concentration, and repeat purchasing behavior using the *theLook eCommerce* dataset.
- Looker Studio was used to illustrate the results, emphasizing executive-level storytelling, clarity, and insight.

## Key Insights Explored
- Revenue growth and momentum over time
- Category-level revenue concentration
- Dependence on high-value and repeat customers
- Long-term customer value vs short-term engagement
  
## Tech Stack Used
- **Data Source:** BigQuery Public Dataset – theLook eCommerce  
- **SQL Engine:** Google BigQuery  
- **Visualization:** Looker Studio  
- **Version Control:** GitHub
